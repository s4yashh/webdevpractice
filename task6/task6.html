Web Programming Foundations

The Web Programming Foundations course introduces students to the core technologies used to build modern websites. This course is designed for beginners who want to start a career in web development.

Topics Covered

  • Text Formatting and Lists
  • Images and Hyperlinks
  • Semantic HTML
  • Basic Web Page Structure
